# AI-Powered Job Portal

## Project Overview

AI-Powered Job Portal is a full-stack web application built using React, Spring Boot, MySQL, and Google Gemini AI.

The platform allows job seekers to search and apply for jobs, recruiters to post and manage jobs, and administrators to monitor the platform.

## Features

### Job Seeker

* User Registration & Login
* Search Jobs
* Apply for Jobs
* Resume Upload
* Profile Management
* Application Tracking

### Recruiter

* Recruiter Registration & Login
* Post Jobs
* Edit Jobs
* Delete Jobs
* View Applicants

### Admin

* User Management
* Recruiter Management
* Job Moderation
* Dashboard Analytics

### AI Features

* Gemini AI Integration
* Resume Analysis
* Skill Recommendations
* Job Recommendations

## Tech Stack

### Frontend

* React.js
* React Router
* Bootstrap
* Axios

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* Spring Data JPA

### Database

* MySQL

### AI

* Google Gemini API

## Installation

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm start
```
